# Solidity Lab Simulator 🧪

Solidity Lab Simulator는 블록체인 스마트 컨트랙트 개발을 위한 [Remix IDE](https://remix.ethereum.org)의 핵심 기능을 웹 브라우저 상에서 시뮬레이션하는 경량화된 인터랙티브 교육용 도구입니다.

이 시뮬레이터는 복잡한 환경 설정이나 실제 메인넷/테스트넷 가스비 지불 없이, 브라우저 상에서 안전하게 솔리디티(Solidity) 코드를 작성, 가상 컴파일, 배포, 그리고 상호작용(Interaction)하는 전체 과정을 실습한 결과입니다.

## 🚀 주요 기능 (Features)

1. **가상 블록체인 환경 (Mock Blockchain Engine)**
   * 가상의 계정 5개와 각 계정당 100 ETH의 초기 잔액을 제공합니다.
   * 트랜잭션 발생 시 가상의 가스비(Gas Fee)가 차감되며, 가상 트랜잭션 해시(TxHash)와 블록 넘버가 생성됩니다.
2. **라이브 코드 에디터 (Interactive Editor)**
   * 솔리디티 문법 하이라이팅(Syntax Highlighting) 지원.
   * 실시간 줄 번호 표시 및 커서 위치 추적 기능.
   * `Tab` 키를 통한 들여쓰기 지원.
   * 탭(Tab) 이동 시 코드 자동 저장 기능 내장.
3. **가상 컴파일러 및 배포기 (Mock Compiler & Deployer)**
   * 입력된 코드에서 상태 변수(State Variables)와 함수(Functions), 그리고 접근 제어자(`public`, `private` 등)를 파싱하여 컨트랙트를 가상으로 컴파일합니다.
   * `Deploy` 버튼 클릭 시, 컴파일된 컨트랙트를 가상 네트워크에 배포하고 인터랙션 패널에 UI를 동적으로 생성합니다.
4. **인터랙션 및 시각화 (Interaction Panel & State Visualizer)**
   * 배포된 컨트랙트의 `public` 변수(Getter)와 함수를 직접 호출할 수 있는 버튼과 입력 폼을 제공합니다.
   * 트랜잭션 실행 시 변환되는 가상 스토리지의 상태(State) 변화를 실시간으로 시각화하여 보여줍니다.
5. **터미널 콘솔 및 알림 (Console & Toasts)**
   * 컴파일 결과, 트랜잭션 해시, 함수 반환값 등 주요 이벤트를 기록하는 하단 터미널 콘솔을 지원합니다.
   * 우측 상단의 Toast 팝업을 통해 직관적인 시스템 알림을 제공합니다.

## 🛠 실행 방법 (How to Run)

본 시뮬레이터는 순수 HTML, CSS, Javascript (Vanilla JS)로 작성되어 별도의 서버나 패키지 설치가 필요하지 않습니다.

1. 본 저장소를 클론(Clone) 하거나 `simulator.html` 파일을 다운로드합니다.
2. 다운로드한 `simulator.html` 파일을 웹 브라우저(Chrome, Edge, Firefox 등)에 끌어다 놓거나, 더블 클릭하여 엽니다.
3. **런처(Launcher) 화면**에서 `Lab 1` 카드를 클릭하거나 숫자 키 `1`을 눌러 실습 환경에 진입합니다.

## 💡 사용 가이드 (User Guide)

1. **계정 선택:** 상단 헤더에서 실습에 사용할 가상 지갑 계정을 선택할 수 있습니다.
2. **코드 작성:** 좌측 '코드 에디터'에서 솔리디티 코드를 작성합니다. 중앙의 탭(1. Hello, 2. Ext., 3. Number)을 눌러 제공된 예제 코드를 불러올 수 있습니다.
3. **컴파일:** 코드를 다 작성한 후, 중앙 패널의 **컴파일** 버튼을 클릭합니다. (단축키: `Ctrl + Enter`)
4. **배포:** 컴파일이 성공하면 **배포** 버튼이 활성화됩니다. 클릭하여 가상 네트워크에 배포합니다. (단축키: `Ctrl + Shift + D`)
5. **인터랙션:** 우측 '인터랙션 패널'에 생성된 버튼들을 눌러 스마트 컨트랙트의 함수를 실행하고 가스비 차감 및 상태 변화를 확인합니다.
6. **콘솔 확인:** 하단의 터미널 콘솔을 열어 자세한 트랜잭션 로그를 확인합니다. (단축키: `Ctrl + \``)

## 🗂 프로젝트 구조 (Structure)

* **CSS Styles**: 네온(Neon) 테마를 적용한 모던한 UI와 에디터 렌더링을 담당합니다.
* **BlockchainEngine**: 블록체인 상태(계정, 잔액, 블록 넘버 등)와 가스비 계산 로직을 모방하는 객체입니다.
* **editor**: 텍스트 영역 이벤트 처리 및 정규식을 활용한 솔리디티 문법 하이라이팅을 수행합니다.
* **compiler**: 솔리디티 코드에서 `contract`, `function`, 변수 등의 패턴을 파싱하여 배포 가능한 가상 데이터 구조를 생성합니다.
* **app**: 전체 애플리케이션의 라우팅, 키보드 단축키, UI 탭 전환 등을 관리합니다.
